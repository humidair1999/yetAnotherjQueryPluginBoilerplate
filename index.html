Goals/Aspirations

  • Namespacing - uses module pattern and notifies when jquery plugin function namespace is occupied
  • Protects against multiple instantiations - DONE
  • jQuery pseudo-selector - DONE
  • Getter/setter for plugin options - DONE
  • Chaining support - up to user when implementing public methods
  • Uses module design pattern (provides private and public methods) - DONE
  • Ability to close/destroy plugin instance - DONE
  • Ability to define optional options or callback - DONE
  • Callback hooks on plugin initialization/destruction - DONE
